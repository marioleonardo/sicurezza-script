// question: 2  name: Quali tra le seguenti affermazioni sulla crittografia simmetrica sono vere?
// [tag:crypto] [tag: simmetrica] 
::Quali tra le seguenti affermazioni sulla crittografia simmetrica sono vere?::[html]<p><strong>Quali tra le seguenti affermazioni sulla crittografia simmetrica sono vere?</strong></p>{
	~%50.00000%[moodle]La crittografia simmetrica richiede una chiave segreta condivisa tra mittente e destinatario per cifrare e decifrare i dati.
	~%50.00000%[moodle]Gli algoritmi simmetrici come AES sono generalmente più efficienti di quelli asimmetrici per grandi volumi di dati.
	~[moodle]La distribuzione sicura delle chiavi nella crittografia simmetrica è facilmente risolvibile senza l’uso di ulteriori protocolli.
	~[moodle]Gli algoritmi di crittografia simmetrica non sono vulnerabili a brute force se la chiave è sufficientemente lunga.
}

// question: 4  name: Quali tra le seguenti affermazioni sul padding nella crittografia a blocchi sono vere?
// [tag:crypto] [tag: simmetrica] 
::Quali tra le seguenti affermazioni sul padding nella crittografia a blocchi sono vere?::[html]<p><strong>Quali tra le seguenti affermazioni sul padding nella crittografia a blocchi sono vere?</strong></p>{
	~%50.00000%[moodle]Il padding garantisce che l’input della cifratura a blocchi sia un multiplo della lunghezza del blocco.
	~%50.00000%[moodle]In CBC, il padding viene aggiunto solo al blocco finale, indipendentemente dalla lunghezza del messaggio.
	~[moodle]Il padding è necessario in tutte le modalità operative della crittografia a blocchi.
	~[moodle]Il padding è sempre utilizzato per garantire integrità dei dati cifrati.
	~[moodle]Algoritmi come AES in modalità CBC non richiedono padding se la lunghezza del messaggio è nota.
}
